Website Deployment Project

Overview

Was created website as part of learning process. Used various DevOps tools like Jenkins, Docker, Git, HTML, CSS to deploy his website to a web server hosted on AWS. The infrastructure was described using Terraform.

AWS Virtual Machines

  • 'jenkins_server'
  • 'jenkins_agent'
  • 'web_server'

Deployment Steps

  1. Developer does a commit to GitHub
  2. GitHub webhook triggers a job in Jenkins Server
  3. Jenkins Server runs stages on Jenkins Agent:
    1. Fetch data from GitHub
    2. HTML code validation
    3. Publish image to Docker Hub
    4. Deploy to Web Server:
      • Stop previous running container
      • Download latest image from Docker Hub
      • Run latest image with name function and ports mapping
      • Filter by tag (latest) and delete container
