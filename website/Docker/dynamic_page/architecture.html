AWS Architecture

Using Terraform to describe infrastructure as code is a good practice and it helps in automating the infrastructure deployment process. Building the project on AWS platform is a provides various services and features that can help in building a scalable and secure infrastructure. Using Docker to launch the site based on Nginx image is it provides a lightweight and portable environment to run the application. Additionally, using CloudFront for the static website is it helps in delivering the content faster to the end-users by caching the content on the edge locations. Jenkins is a popular CI/CD tool and using it in the project is it helps in automating the build, test and deployment process. Using Public Subnets to avoid additional costs for NAT Gateway is a good approach to minimize the costs, however, it's important to consider the security aspects while using Public Subnets.
