My DevOps Journey

In June 2022, I started my DevOps training with the skills of a normal user.

My friend, Mr. DevOps, offered me to study and we were disappointed. He made a clear plan for me, which included:

  • Learning about networks as a basis for future learning
  • Learning the Linux operating system
  • Creating my working repository in Git
  • Learning how to use the command line with various shells
  • Taking online video courses on AWS

Later, we described the infrastructure that would allow me to create my own site on AWS platform using Terraform. In it, we used Docker and Jenkins for CI/CD. The website runs on Nginx. With the help of CloudFront, we also made a static version of the site.

I also independently took courses in Git, Linux, Docker, and improved my English level. I would really like to find a job in this new field for me, develop on a real project, and gain new knowledge.
